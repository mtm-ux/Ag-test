Prototyp Live

Level 1: Injection Point

> Ziel: Atmosphärisches, narratives Typing-Game im Cyberpunk/Hacker-Stil

Tech-Stack: HTML5 · CSS3 (Glitch-Effekte, CRT-Scanlines) · Vanilla JS | Zielgruppe: Noah NoHands (Gamer, Story & Immersion)

Lernziel

Grundreihe (ASDF JKLÖ) – der Operator meistert die Basis-Eingabe

Narrativer Hook

"Verbindung instabil. Firewall aktiv. Initialisiere manuellen Override."

Visueller Stil

Dunkles Terminal (Schwarz/Dunkelgrau), Grüne Schrift, CRT-Scanlines, RGB-Shift-Flimmern

1. Das Setting & Die Story

Der Bildschirm ist kein "Spiel" – es ist ein Terminal. Noah ist ein neuer Rekrut ("Operator"), der versucht, eine Verbindung zum abgeschotteten Server herzustellen, bevor das System ihn aussperrt.

> INITIATING HANDSHAKE...
> ERROR: AUTO-PILOT FAILED.
> MANUAL INPUT REQUIRED.
> OPERATOR, TAKE CONTROL.

2. Visuelle Architektur (UI/UX)

A. Der Hintergrund (The Void)

  • • CRT-Overlay: CSS-Layer mit feinen horizontalen Linien (pointer-events: none) – Röhrenmonitor-Simulation
  • • Background-Animation: Dunkle Hexagon-Fragmente oder Code-Schnipsel regnen im Hintergrund (Matrix-Style, minimalistisch, in Dunkelgrün)

B. Das HUD (Head-Up Display)

  • • Status-Leiste (Oben):
  •   Links: SYS_INTEGRITY: 100% (sinkt wenn inaktiv)
  •   Rechts: USER: NOAH_N0H4NDS | PING: 24ms
  • • Core-Terminal (Mitte): Großes rahmenloses Eingabefeld, blinkender Block-Cursor (_)
  • • Target-Code anfangs "verschlüsselt" (zufällige Zeichen), stabilisiert sich zum lesbaren Buchstaben

C. Die Bedrohung: The Corruption Bar

Roter Balken am unteren Bildschirmrand, füllt sich langsam. Jeder richtige Anschlag drückt ihn zurück. Ist er voll → "CONNECTION TERMINATED" (Game Over).

3. Game-Logik & Level-Phasen

INIT
Intro-Sequenz (Scripted)

Schwarzer Schirm. Text tippt sich automatisch ein (Typewriter-Effekt):

> INITIATING HANDSHAKE...
> ERROR: AUTO-PILOT FAILED.
> MANUAL INPUT REQUIRED.
> OPERATOR, TAKE CONTROL.

PHASE 1
Der "Ping" – Einzelne Tasten

Das System sendet einzelne Datenpakete. Noah muss sie bestätigen.

Display: [ _ ] – Zielbuchstabe flackert kurz auf → A ... S ... D ... F
Feedback: Bei Erfolg wird der Buchstabe hellweiß und "rastet ein" mit mechanischem Sound.

PHASE 2
Der "Brute Force" – Kombinationen

Story: "Firewall entdeckt. Doppelte Injektion nötig."

Buchstabenpaare tippen: [ A S ] oder [ J K ]
Mechanik: Der rote Corruption Bar füllt sich jetzt schneller (Druckerhöhung).

PHASE 3
Der "Payload" – Wörter & Code

Story: "Sicherheitslücke gefunden. Lade Payload hoch."

Echte Wörter oder Pseudo-Code-Fragmente: DAS FALL LAD SAD GLAS
Finale: Das Wort ACCESS (oder START falls nur Grundreihe bekannt).

4. Animationen & Effekte ("Das Juice")

Der "Glitch" (Bei Fehler)

  • • Screen verschiebt sich horizontal (CSS transform: translate)
  • • Farben invertieren für 0.1s (Weißer Blitz)
  • • Sound: Verzerrtes Modem-Kratzen / Static Noise
  • • Text-Overlay: > PACKET LOSS DETECTED

Der "Flow" (Bei Success)

  • • Buchstabe leuchtet Neon-Grün auf
  • • Hinterlässt beim Verschwinden einen "Ghost-Trail"
  • • Sound: Sattes, tiefes mechanisches Klacken (IBM Model M)
  • • Visueller Confirmation-Pulse

5. Easter Egg: "God Mode"

Auslöser: Wenn im Terminal kein aktives Wort steht (z.B. zwischen zwei Phasen oder während des Intros) und Noah blind das Wort tippt: H A C K

> ROOT ACCESS GRANTED. GOD MODE ACTIVE.

  • • Visual Change: Farbschema wechselt von Grün/Schwarz zu Gold/Schwarz
  • • Gameplay-Vorteil: Corruption Bar friert 10 Sekunden ein (Unsterblichkeit)
  • • Sound: Tiefe, göttliche Synth-Stimme: "ACCESS GRANTED"
